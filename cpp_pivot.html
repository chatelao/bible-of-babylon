Cpp Pivot View

Cpp Pivot Table

Pattern

Syntax

Notes

MutexDefinition

 
std::mutex mtx;

Requires <mutex> header (C++11).

MutexLock

 
mtx.lock();

Better handled with std::lock_guard for RAII.

MutexUnlock

 
mtx.unlock();

Releases the lock.

SemaphoreDefinition

 
std::counting_semaphore<10> sem(1);

Requires <semaphore> header (C++20).

SemaphoreWait

 
sem.acquire();

Decrements and blocks if needed.

SemaphoreSignal

 
sem.release();

Increments the semaphore count.

VariableDeclaration

 
int x = 42;

Static typing, similar to C.

CollectionDefinition

 
std::vector<int> v = {1, 2, 3};

Standard vectors are the common dynamic collection in C++.

AssociativeArrayDefinition

 
std::map<std::string, int> m = {{"a", 1}, {"b", 2}};

The map or unordered_map classes provide associative storage.

Equal

 
a == b

Standard equality operator.

NotEqual

 
a != b

Standard inequality operator.

GreaterThan

 
a > b

Standard greater than operator.

LogicalAnd

 
a && b

Short-circuiting logical AND.

LogicalOr

 
a || b

Short-circuiting logical OR.

LogicalXor

 
a != b

C++ does not have a dedicated logical XOR operator; for booleans, != serves as XOR.

ProcedureDefinition

 
void logMessage(std::string msg) {
    std::cout << msg << std::endl;
}

Procedures use the ‘void’ return type.

FunctionDefinition

 
int add(int a, int b) {
    return a + b;
}

Standard C-style function definition.

IfElse

 
if (x > 0) {
    return 1;
} else {
    return 0;
}

Standard C-style if-else statement.

SwitchCase

 
switch (x) {
    case 1:
        return 1;
    case 2:
        return 2;
    default:
        return 0;
}

Standard C-style switch statement.

Loop

 
while (x > 0) {
    x = x - 1;
}

Standard C-style while loop.

ForLoop

 
for (int i = 0; i < 10; i++) {
    // body
}

Standard C-style for loop.

ForEach

 
for (const auto& item : collection) {
    // body
}

The range-based for loop introduced in C++11.

Addition

 
a + b

Standard arithmetic operator.

Subtraction

 
a - b

Standard arithmetic operator.

Multiplication

 
a * b

Standard arithmetic operator.

Division

 
a / b

Standard arithmetic operator.

Remainder

 
a % b

Standard arithmetic operator.

Floor

 
std::floor(a)

Requires <cmath>.

Rounding

 
std::round(a)

Requires <cmath>.

Increment

 
a++

Standard increment operator.

Decrement

 
a--

Standard decrement operator.

LeftShift

 
a << b

Standard bitwise shift operator.

RightShift

 
a >> b

Standard bitwise shift operator.

BitAnd

 
a & b

Standard bitwise operator.

BitOr

 
a | b

Standard bitwise operator.

BitXor

 
a ^ b

Standard bitwise operator.

BitNot

 
~a

Standard bitwise operator.

TryCatch

 
try {
    do_something();
} catch (const std::exception& e) {
    handle(e);
}

Standard C++ exception handling using try-catch blocks.

Raise

 
throw std::runtime_error("Error");

Uses ‘throw’ to raise an exception.

Thread

 
std::thread t(do_work);
t.join();

Uses std::thread (C++11 and later).

SendMessage

 
queue.push(42);

Communication is typically handled via shared data structures and synchronization primitives.

ReceiveMessage

 
int msg = queue.front();
queue.pop();
handle(msg);

Typically involves pulling from a thread-safe queue.

SingleLineComment

 
// comment

Standard C++ single-line comment.

MultiLineComment

 
/* line 1
   line 2 */

Standard C++ multi-line comment.

Print

 
std::cout << "Hello, World!" << std::endl;

Uses std::cout for standard output; requires <iostream>.

Import

 
#include <iostream>

Standard way to include headers.

Constant

 
const int MAX = 100;

The ‘const’ keyword defines an immutable value.

Float4VectorMultiplication

 
for (int i = 0; i < 4; i++) c[i] = a[i] * b[i];

Standard C++ requires a loop or explicit component-wise multiplication for arrays.

Float4VectorDotProduct

 
float dot = 0; for (int i = 0; i < 4; i++) dot += a[i] * b[i];

Calculated by accumulating component-wise products in a loop.

Float4VectorCrossProduct

 
c[0] = a[1]*b[2] - a[2]*b[1];
c[1] = a[2]*b[0] - a[0]*b[2];
c[2] = a[0]*b[1] - a[1]*b[0];
c[3] = 0.0f;

Manual implementation for the first three components.

SetFiltering

 
std::copy_if(v.begin(), v.end(), std::back_inserter(res), condition);

Uses the standard library algorithm copy_if.

SetJoin

 
std::set_intersection(v1.begin(), v1.end(), v2.begin(), v2.end(), std::back_inserter(res));

Uses set_intersection for sorted ranges; otherwise uses nested loops.
